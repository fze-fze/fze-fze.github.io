# 工作流总览



## 软件类

**个人管理**：备忘录、提醒事项、日历、fze's blog

**学习平台**：ChatGPT、Goodnotes、不背单词、Typora、VS Code

**信息渠道**：播客、X、公众号、B站、YouTube

**校园平台**：企业微信、QQ、微信、刷课平台



## 时间线

**Deep** 线：专注学习模式（40min 一轮），完整记录完成内容

**Maintain** 线：日常事物/杂项处理，**通勤**时集中处理

**Habit** 线：每天需要完成的任务，一般为固定项目

**Explore** 线：暂时没有时间了解的新知识，以后探索



## 工作块

**睡前块**：睡前总结、冥想，**睡眠模式**

**通勤块**：上下课、吃午饭，听播客、音乐、摄入信息

**上课块**：学校课程/会议/比赛，可以刷课，**专注模式**

**学习块**：40min 不间断学习，**专注模式**

# 管理流



## 备忘录

### 结构示意图

```
iCloud/
├── 快速备忘录/ # 随手记录信息
│ 
├── 备忘录/ # 记载马上要用的急切文件/事情
│    
├── ⚙️提醒事项/	# 智能文件夹，直接从提醒事项跳转	
│    
├── 2026年/ # 本年度的管理文件夹
│   ├── 26春季/ # 校内课程的规划 
│   ├── 目标/ # 想自学的一些技能/知识
│   ├── 记录/ # 写的一些临时文章/随笔
│   └── 文件/ # 不一定要用的文件夹
│    
└── 最近删除/ # 保质期一个月
```

### IO系统

输入流：

- **通勤块** 从 **信息渠道** 获取灵感，进入 **快速备忘录**
- **学习块** 在没有电脑的情况下写文章/随笔，进入 **2026年/记录**
- **通勤块** 从 **校园平台** 获取作业/通知，进入 **快速备忘录** 

输出流：

- **专注块** 从 **快速备忘录** 找到作业/通知，进入 **提醒事项**
- **专注块** 从 **快速备忘录** 找到灵感，写文章/随笔，进入 **2026年/记录**
- **睡前块** 从 **2026年/记录** 导出文章，进入 **fze's blog**



## Blog

### 文字结构

```
fze's journal/
├── article/ # 长篇深度思考文章
│   ├── About Me🔝
│   └── ...
│   
├── essays/ # 短篇灵感随笔
│ 
└── plans/ # 每周透明计划
```

### 维护思路

- 文章、随笔：思考后的产出结果

- 计划：以周为单位，规划一周的学习、工作

    每周日进行复盘/规划，确定 3 个每周主线任务
